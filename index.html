    Description

  • Salope 3.0

    Description

    Perdu dans l'ombre, je me perds en moi-même, Les nuages noirs obscurcissent ma vue déjà trouble. Les mots se bousculent, prisonniers dans ma gorge, La beauté ou la laideur, en moi, un mystère déroutant.

  • Salope 4.0

    Description

    Entre les lignes de ma peine et de ma joie tordue, Je cherche la paix, l'harmonie entre deux extrêmes. Mon chagrin, une mélodie de beauté délicate, Mon bonheur, une danse étrange de laideur assumée.
